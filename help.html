JobTrack

[ Home ]

JobTrack is an extension aimed to keep saved job postings organized in one place and increase productivity by motivating users to fill out their associated applications.

How to Use JobTrack

  1. Navigate to a job posting of interest to save/note
  2. Using the text box, type the description of the posting (ex. the company and job title of the posting)
  3. JobTrack will add that description to the list, linked to the webpage of the corresponding job posting
  4. Once a user completes a job application for a certain job posting, they can check the box next to the description and press the DONE button to remove it
  5. To clear the entire list, DOUBLE CLICK the CLEAR ALL button
